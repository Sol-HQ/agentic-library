# Agentic Library

A curated, agent-readable library of high-signal open-source resources for building AI agents, memory systems, voice interfaces, and automation stacks.

This repository is designed for two audiences:

- Humans: a practical learning path and implementation playbooks.
- Agents: structured dossiers and catalogs that can be ingested as context.

## Why this exists

You asked for a dedicated GitHub repository that acts like a world-class learning institution for agentic systems, with concrete resources that can be reused across:

- Trading bot projects
- Minting/NFT projects
- Solana + agent infrastructure work
- Future product experiments

## Repository layout

- `catalog/`: machine-readable and comparison data.
- `knowledge/repo-dossiers/`: one dossier per tracked project.
- `knowledge/tracks/`: learning tracks and progression.
- `sol-hq-alignment/`: where each resource maps to current Sol-HQ repos.
- `playbooks/`: execution plans for adoption and rollout.
