# Agent Engineering Curriculum

A practical reading and build sequence designed for your Agentic Library.

## Track 1: Foundations (Week 1)

1. `andrej-karpathy-skills.md`
2. `open-agents.md` (architecture sections)
3. `magika.md`

Outcome: team adopts consistent agent behavior principles and baseline safety patterns.

## Track 2: Memory and Context (Week 2)

1. `claude-mem.md`
2. `cognee.md`
3. `evolver.md`

Outcome: define your memory stack strategy and controlled optimization loop.

## Track 3: Product Interfaces (Week 3)

1. `voicebox.md`
2. `omi.md`
3. `kreo.md` (benchmark only)

Outcome: identify user-facing experiences (voice, multimodal context, trading signals).

## Track 4: Autonomous Execution Patterns (Week 4)

1. `genericagent.md`
2. `open-agents.md` (runtime execution model revisit)
3. Internal design review against Sol-HQ roadmap

Outcome: decide hosted vs local execution architecture and governance model.

## Graduation deliverables

- A Sol-HQ `AGENT_RULES.md` baseline.
- A memory architecture doc (capture, compression, retrieval).
- A signed-off integration backlog with owner and timeline.
